# ReactChatFe

## 개요
해당 프로젝트는 React를 사용하여 프론트엔드를 구현한다.
api 서버는 Spring Boot로 구현되어 있다.
Spring Boot 를 통해 websocket and stomp를 사용하여 실시간 채팅 기능을 구현한다.
websocket 기술에 대한 POC를 목적으로 진행된 프로젝트이다.
